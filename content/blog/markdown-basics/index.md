# Язык разметки Markdown

Markdown — это простой язык разметки для создания форматированного текста. Он идеально подходит для заметок, документации и блогов.

## Заголовки

Используйте `#` для заголовков:

```markdown
# Заголовок 1

## Заголовок 2

### Заголовок 3
```

## Списки

Ненумерованный список:

- Пункт 1
- Пункт 2

Нумерованный список:

1. Первый
2. Второй

## Жирный и курсив

- **жирный текст**
- _курсив_
- **_жирный и курсив_**
